## 适用范围

适用于 Windows 办公电脑无法打印、连接不上打印机、打印驱动异常、共享打印机不可用等问题。

## 常见现象

- 打印任务无反应。
- 连接不上共享打印机。
- 打印机驱动故障。
- 更换工位或人员调动后无法打印。
- 打印模板或纸张规格不正确。

## 初步判断

先确认打印机本体是否正常，再判断是电脑驱动、共享地址、权限、网络还是模板问题。

## 处理步骤

1. 确认打印机电源、网线和面板状态正常。
2. 检查是否只有单台电脑无法打印。
3. 清空本机打印队列。
4. 重新连接正确的共享打印机或打印服务器地址。
5. 必要时卸载旧驱动后重新安装驱动。
6. 如果是模板、纸张规格或标识卡打印问题，检查应用模板和纸张设置。
7. 打印测试页确认恢复。

## 验证方式

- 本机能成功打印测试页。
- 业务系统或办公软件能正常输出正式文档。

## 风险点

- 不要删除其他用户正在使用的共享打印机配置。
- 修改打印服务器地址前需要确认正确地址。

## 升级条件

- 多台电脑无法连接同一打印机。
- 打印机面板报硬件故障。
- 打印服务器不可访问。

- [ ] 补充常见打印机型号驱动位置。
