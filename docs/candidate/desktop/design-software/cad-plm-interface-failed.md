## 常见现象

- CAD 与 PLM 接口安装失败。
- CAD 无法保存到管理系统。
- PLM 与 CAD / SolidWorks 接口无法传图。

## 处理步骤

1. 确认 CAD、SolidWorks 和 PLM 客户端版本。
2. 确认接口插件是否安装。
3. 检查用户 PLM 权限和登录状态。
4. 检查接口插件配置和本地缓存。
5. 用测试图纸验证上传或保存。

## 升级条件

- PLM 系统卡顿或接口服务异常。
- 插件版本不匹配。
- 涉及批量设计人员无法传图。
