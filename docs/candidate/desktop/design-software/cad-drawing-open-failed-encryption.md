## 常见现象

- DWG 图纸打不开。
- 提示文件损坏。
- 加密未掉但图纸无法打开。
- 加密客户端版本相关。

## 处理步骤

1. 确认单个图纸异常还是多份图纸异常。
2. 复制文件副本后再尝试修复。
3. 检查加密客户端状态和版本。
4. 如历史记录显示版本问题，升级加密客户端。
5. 使用备份文件或历史版本恢复。

## 风险点

- 不要直接覆盖原图纸。
- 重要图纸先备份再处理。
