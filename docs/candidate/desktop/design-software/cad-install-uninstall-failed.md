## 常见现象

- CAD 装不上、卸不掉。
- 重装 CAD 失败。
- 注册表残留导致安装失败。
- 中文语言包或安装组件异常。

## 处理步骤

1. 确认 CAD 版本和授权方式。
2. 卸载现有 CAD 及相关组件。
3. 清理残留安装目录和注册表项。
4. 使用公司指定安装包重新安装。
5. 完成授权或激活后打开测试图纸。

## 风险点

- 清理注册表前需要确认范围。
- 不要使用未知来源安装包和激活工具。
