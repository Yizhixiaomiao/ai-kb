## 常见现象

- SolidWorks 无法打开或无法卸载安装。
- 提示找不到 `solidworks.msi`。
- 安装源被删除后插件无法找到。

## 处理步骤

1. 确认当前 SolidWorks 版本。
2. 找到对应版本的公司标准安装源。
3. 修复安装源路径或执行修复安装。
4. 如无法修复，完整卸载后重新安装。
5. 重新激活并打开测试模型。

## 升级条件

- 注册表信息无法更改。
- 需要重装系统才能恢复安装环境。
