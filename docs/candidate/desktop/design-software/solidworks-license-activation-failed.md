## 常见现象

- SolidWorks 提示无法获得许可。
- 重启后要求重新激活。
- 激活服务证书或许可证丢失。
- 杀毒软件拦截激活服务证书。

## 处理步骤

1. 确认用户是否有 SolidWorks 使用权限。
2. 检查许可证服务或本地激活状态。
3. 检查安全软件是否隔离许可证相关文件。
4. 恢复被隔离文件或重新激活。
5. 打开 SolidWorks 验证。

## 升级条件

- 许可证服务器异常。
- 多名用户同时无法获取许可。
- 需供应商处理授权。
