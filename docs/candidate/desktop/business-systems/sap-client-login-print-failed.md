## 常见现象

- SAP 登录不上。
- SAP 客户端版本不兼容。
- SAP 界面模糊。
- SAP 调用本地打印无响应。
- SAP 内没有需要的纸张选项或打印代码异常。

## 处理步骤

1. 确认 SAP 客户端版本和目标系统。
2. 检查账号、密码和登录配置。
3. 对版本不兼容问题，安装公司指定版本客户端。
4. 检查 SAP 本地打印配置和默认打印机。
5. 检查纸张规格、SAP 打印代码和业务单据模板。
6. 本地打印正常但 SAP 无响应时，升级给 SAP / 应用管理员。

## 升级条件

- 多名用户 SAP 登录异常。
- SAP 业务单据或打印代码异常。
- 涉及业务系统配置或权限。
