## 常见现象

- 系统崩溃或无法启动。
- 垃圾软件太多，反复弹窗。
- 常用软件闪退，需要重装系统。
- 清空缓存后恢复。

## 处理步骤

1. 判断是否可以通过清理缓存、修复系统或卸载异常软件恢复。
2. 备份用户桌面、文档和业务数据。
3. 记录需要恢复的软件、打印机、共享盘和业务客户端。
4. 按公司标准镜像或流程重装系统。
5. 恢复用户数据和常用软件。
6. 验证网络、打印、办公软件和业务系统。

## 风险点

- 未备份数据前不要重装系统。
- 涉及授权软件时需确认授权方式。

## 升级条件

- 系统盘或硬盘疑似故障。
- 重要数据需要恢复。
