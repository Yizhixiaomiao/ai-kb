# 打印模板或纸张规格错误

```yaml
status: usable

asset_types: [pc, printer]
systems: [windows, oa, wms]
issue_types: [printer-not-working]
tags: [printer, template, paper-size]

```

## 常见现象

- 英文版标识卡无法打印。
- 三联纸只能按 A4 打印。
- 发车清单、库位码或业务单据打印格式异常。

## 处理步骤

1. 确认业务系统、单据类型和打印模板。
2. 检查打印机纸张规格设置。
3. 检查业务系统或浏览器打印设置。
4. 重新设置模板或打印插件。
5. 用测试单据验证。

## 升级条件

- 模板由业务系统控制。
- WMS、OA 或其他业务插件异常。
