## 常见现象

- 打印机驱动故障。
- 误删除打印机后无法重新安装。
- 找不到对应型号驱动。
- 安装错误驱动后无法打印。

## 处理步骤

1. 确认打印机型号和连接方式。
2. 卸载错误或残留驱动。
3. 从公司指定驱动库或打印服务器安装正确驱动。
4. 重新添加打印机。
5. 打印测试页。

## 风险点

- 不要从非可信网站下载驱动。
- 不要误删其他正常打印机驱动。
