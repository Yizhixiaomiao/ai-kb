## 常见现象

- 打印机突然无法使用。
- 打印端口被删除。
- 安装 360 或安全软件后打印异常。

## 处理步骤

1. 确认打印机端口是否存在。
2. 检查近期是否安装或清理过安全软件。
3. 恢复或重新创建打印端口。
4. 必要时卸载冲突安全软件。
5. 重新安装打印机驱动并打印测试页。

## 风险点

- 卸载安全软件前需确认公司终端安全策略。
