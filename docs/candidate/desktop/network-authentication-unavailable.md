## 适用范围

适用于办公电脑、车间终端或外网笔记本出现无网络、认证失败、MES 登不上、网页无法访问等问题。

## 常见现象

- 显示无网络、网络不通、连不上网。
- MES、OA 或内部平台无法打开。
- 认证助手登录失败、认证过多、绑定设备超出数量。
- 重启电脑或插拔网线后仍无法恢复。

## 初步判断

优先区分是单台电脑问题、账号认证问题、网线 / 交换机端口问题，还是业务系统本身异常。

## 处理步骤

1. 确认同区域其他电脑是否正常上网。
2. 检查网线、网口、无线连接状态。
3. 重新打开认证客户端，确认账号、密码和绑定设备数量。
4. 如果提示认证过多或设备超限，清理旧绑定或联系账号管理员处理。
5. 如果只有 MES 或内部平台无法访问，确认浏览器收藏地址是否正确。
6. 必要时重新认证、刷新 DNS 或重启电脑。
7. 如果同一区域多台设备无网络，升级给网络工程师排查交换机、线路或上级网络。

## 验证方式

- 能正常打开内部平台或 MES。
- 认证客户端状态正常。
- 浏览器能访问常用内外网地址。

## 风险点

- 不要随意更换 IP 或网关，避免造成 IP 冲突。
- 账号设备绑定清理需要确认用户身份。

## 升级条件

- 同一区域多台电脑同时无网络。
- 认证平台异常或大量用户无法认证。
- 工位网口无链路灯或疑似交换机端口故障。

## 工程师复核项

- [ ] 补充本公司认证客户端名称和入口。
- [ ] 补充 MES / OA / 内部平台正确访问地址。
- [ ] 明确设备绑定清理权限归属。
