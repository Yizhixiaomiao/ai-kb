## 常见现象

- PDF 文件打不开。
- 提示应用程序并行配置不正确。
- PDF 阅读器闪退或弹出检测错误。
- PDF 打印很慢，提示扁平化处理。
- 其他文件能打印，只有 PDF 不能打印。

## 处理步骤

1. 确认是单个 PDF 异常还是所有 PDF 异常。
2. 修改 PDF 默认打开方式。
3. 修复或重装 PDF 阅读器。
4. 检查安全软件是否隔离 PDF 阅读器依赖文件。
5. 如 PDF 打印慢，检查是否勾选“作为图像打印”等设置。
6. 用测试 PDF 打开和打印验证。

## 风险点

- 重要 PDF 文件先复制备份再处理。

## 升级条件

- PDF 文件疑似损坏且无备份。
- 批量用户 PDF 打印异常。
