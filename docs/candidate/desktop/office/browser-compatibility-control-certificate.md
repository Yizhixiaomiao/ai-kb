```yaml
status: usable

```

## 常见现象

- 业务网站运行不了。
- IE 兼容性视图需要调整。
- 控件异常。
- 证书到期导致页面或软件打不开。

## 处理步骤

1. 确认业务系统要求的浏览器版本。
2. 设置兼容性视图或可信站点。
3. 安装或修复必要控件。
4. 检查证书是否过期或被安全软件拦截。
5. 重新打开页面验证。

## 升级条件

- 证书更新涉及服务器或系统侧。
- 多名用户同一业务页面异常。
