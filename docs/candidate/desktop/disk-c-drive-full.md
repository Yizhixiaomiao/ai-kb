## 适用范围

适用于 Windows 电脑 C 盘空间不足、系统盘爆满、因磁盘空间不足导致软件打不开等问题。

## 常见现象

- C 盘可用空间过小。
- Excel、Office 或其他软件无法打开。
- 系统提示磁盘空间不足。
- 电脑明显卡顿。

## 处理步骤

1. 查看 C 盘剩余空间。
2. 使用系统磁盘清理工具清理临时文件、回收站和更新缓存。
3. 检查桌面、下载、微信 / 企业微信 / 浏览器缓存是否占用过大。
4. 将非系统业务文件迁移到其他盘或指定共享位置。
5. 如分区容量确实过小，评估是否扩容 C 盘。
6. 重启电脑后确认软件是否恢复正常。

## 验证方式

- C 盘剩余空间恢复到可接受范围。
- 原异常软件能正常打开。

## 风险点

- 不要删除用户文件和业务数据。
- 扩容分区前需确认数据备份。

## 升级条件

- 硬盘疑似坏道或系统盘损坏。
- 无法释放足够空间。
- 需要调整分区或更换硬盘。
