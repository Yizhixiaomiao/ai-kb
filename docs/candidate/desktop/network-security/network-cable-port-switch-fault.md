## 常见现象

- 网络不通。
- 连接不上网络。
- 网线插上不亮。
- 以太网打叉。
- 墙上网口接触不良。
- 某区域摄像头、门禁或终端不通。

## 处理步骤

1. 确认是否单台设备异常还是同区域多台异常。
2. 检查网线是否松动、破损。
3. 更换网线或网口交叉验证。
4. 检查墙插模块和交换机端口指示灯。
5. 如同区域多设备异常，检查上联交换机、电源或光纤链路。
6. 记录位置后升级网络工程师。

## 升级条件

- 墙插、模块或交换机端口故障。
- 同区域多台设备网络不通。
- 涉及摄像头、门禁、道闸等生产或安防设备。
