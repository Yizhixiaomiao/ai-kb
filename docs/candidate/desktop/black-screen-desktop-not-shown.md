## 适用范围

适用于 Windows 登录后黑屏、只有鼠标光标可见、桌面和任务栏不显示等问题。

## 常见现象

- 屏幕黑屏，但鼠标能移动。
- 桌面、任务栏不显示。
- 重启后偶发恢复或反复出现。

## 处理步骤

1. 按 `Ctrl + Shift + Esc` 打开任务管理器。
2. 选择“运行新任务”。
3. 输入 `explorer.exe` 并运行。
4. 如果桌面恢复，检查是否存在系统更新、启动项异常或配置过低。
5. 清理不必要启动项和垃圾软件。
6. 如果频繁复现，升级给系统工程师进一步排查用户配置文件或系统组件。

## 验证方式

- 桌面和任务栏恢复显示。
- 注销或重启后能正常进入桌面。

## 风险点

- 不要直接删除用户配置文件。
- 涉及重要桌面文件时先确认数据位置。

## 升级条件

- 无法打开任务管理器。
- `explorer.exe` 无法启动。
- 多次重启后问题仍复现。
