套料系统部署手册

将SQL SERVER 2016拷贝到服务器并安装（en_sql_server_2016_enterprise_x64_dvd_8701793.iso为数据库服务、SSMS-Setup-ENU.exe数据库管理工具），如果使用现有的数据库，此步骤可以忽略。

运行gbox.exe后输入许可号进入

配置数据库

点击启动，安装主程序（当中会弹出外部程序，需要点击下一步安装即可）

安装VC++组件和加密锁驱动

点击访问按钮，即可访问程序，进入admtools页面初始化数据库（数据库初始化后，如果连接不上，可以打开gbox，重启一下平台服务）
