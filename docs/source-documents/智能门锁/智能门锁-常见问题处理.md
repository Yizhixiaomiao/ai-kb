## 原始正文

应用系统运行维护

1. 系统关闭/启动

(1) tomcat停机服务

远程应用服务器<IP地址已脱敏>，执行如下命令/home/tomcat/bin/shutdown.sh

(2) tomcat启动服务

远程应用服务器<IP地址已脱敏>，执行如下命令/home/tomcat/bin/startup.sh

（3）数据库启动服务

远程应用服务器<IP地址已脱敏>，执行如下命令service mysql restart

2. 应用状态监控

应用的访问日志的访问方法

/home/tomcat/logs

使用命令打开

Tail -f catalina.out

3. 数据说明

(1) 应用服务器日志数据

记录应用服务启动，后台输出等，保存路径为：/home/tomcat，日志按照日生成文件

注：由于应用服务器严格限制磁盘空间大小，如果生成的日志量较大，建议存放在文件服务器上

(3) 数据库

数据库文件保存路径为：

/usr/local/mysql/data/

4. 数据清理

应用日志可以按照日期清理,操作方式：

远程到应用服务器

/home/tomcat/logs

按文件日期删除日志文件

5. 常见问题及处理

磁盘存储空间不足，执行如下命令：true > /home/tomcat/logs/catalina.out
