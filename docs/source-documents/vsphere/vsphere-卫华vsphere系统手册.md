系统概述

VMware vSphere 是 VMware 的虚拟化平台，可将数据中心转换为包括 CPU、存储和网络资源的聚合计算基础架构。vSphere 将这些基础架构作为一个统一的运行环境进行管理，并为您提供工具来管理加入该环境的数据中心。

vSphere 的两个核心组件是 ESXi 和 vCenter Server。ESXi 是用于创建并运行虚拟机和虚拟设备的虚拟化平台。vCenter Server 是一项服务，用于管理网络中连接的多个主机，并将主机资源池化。

关闭和启动 Dell SC 系列存储

关闭

➊ 确保前端所有服务器已关闭，存储无I/O访问

➋ 单击右上角的 Actions

➌ 从显示的菜单中选择 System->Shutdown/Restart。

➍ 在关闭/重新启动窗口中，从“What should the Storage center do?”下拉菜单中选择 Shutdown，然后单击 OK。

➎ 等待大约 5 分钟，检查确认存储管理IP不能访问（此时前后面板指示灯常亮，风扇正常运转）

➏ 关闭存储系统背面左侧和右侧的电源开关。如果连接了磁盘扩展机箱，也请关闭扩展机箱背面左侧和右侧的电源开关。

（如果没有电源开关，请拔下交流电源线。）

启动

如果连接了磁盘扩展机箱，请先开启磁盘扩展机箱电源，等待3到5分钟再打开主柜电源开关。

➊ 请打开电源开关。

电源开关打开后，会开始启动 5 到10分钟（具体取决于存储系统型号）。

➋ 打开 Dell Storage Manager 并检查是否可以登录到存储系统。

➌ 检查存储系统是否正常，然后启动前端服务器。
