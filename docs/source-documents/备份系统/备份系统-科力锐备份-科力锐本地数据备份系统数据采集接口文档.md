# 科力锐本地数据备份系统据采集接口

### 0、接口综述与认证

- 接口类型:
    - HTTP 端口 80或8000
    - HTTPS 端口 443

- 认证（使用 sec_token 机制）:
    - sec_token 为字符串，长度为32个字符。例如：`api_sec_token_b6689f31c8442`
    - 所有科力锐私有云，将内置统一的 sec_token
    - 调用该章节所述接口时，需要在url中加入 `?token=<已脱敏> 字段

<div STYLE="page-break-after: always;"></div>
### 1、资源类型统计

- url: `https://<IP地址已脱敏>/apiv1/v1/resources/?token=<已脱敏>
- method: GET
- response:

```json5
{
  // 操作用户提示信息
  "message": "获取成功",
  // 错误码：0为成功，非0为失败 
  "errcode": 0,
  "data": {
    resource: [
      {
        // 客户端标识
        "hostIdent": "host_123456",
        // 客户端名称
        "clientName": "TEST-3",
        // 客户端类型
        "clientType": "普通客户端",
        // ip
        "ip": "<IP地址已脱敏>",
        // 数据传输方式： "加密" / "不加密"
        "transportMethod": "不加密",
        // 客户端状态:  "在线" / "离线"
        "status": "离线",
        // 最近在线时间
        "lastLinkTime": "2021-01-17 11:19:49",
        // 所属分组
        "group": "未分组"
      },
      ...
    ]
  }
}
```

### 2、备份运行详情

- url: `https://<IP地址已脱敏>/apiv1/v1/backup/work/?token=<已脱敏>
- method: GET
- response:

```json5
{
  "message": "获取成功",
  "errcode": 0,
  "data": {
    resource: {
      // 客户端标识
      "hostIdent": "host_123456",
      // 客户端名称
      "clientName": "TEST-1",
      // 任务类型: "整机CDP" / "整机备份"
      "clientType": "整机CDP",
      // 任务开始时间: "" / "2021-01-25 19:52:29" (当为空字符串意为没有执行过任务)
      "startTime": "2021-01-25 19:52:29",
      // 任务状态: "没有备份计划" / "等待执行" / "进行中" / "备份成功" / "CDP保护中" / "等待重试" / "备份失败"
      "status": "CDP保护中",
      // 任务详情
      "detail": ""
    }
  }
}
```

### 3、客户端空间使用量

- url: `https://<IP地址已脱敏>:80/apiv1/v1/clients/spaceusages/?token=<已脱敏>
- method: GET
- response:

```json5
{
  "message": "获取成功",
  "errcode": 0,
  "data": {
    // 存储节点空间使用信息
    storages: {
      // 存储节点(storage_123456)
      "storage_123456": {
        // 存储节点名称
        "storageName": "存储节点1",
        // 已用空间，单位GB
        "used": 235,
        // 总空间，单位GB
        "total": 474,
      },
      // 存储节点(storage_abcdef)
      "storage_abcdef": {
        // 存储节点名称
        "storageName": "存储节点1",
        // 已用空间，单位GB
        "used": 235,
        // 总空间，单位GB
        "total": 474,
      },
      ...
    },
    // 客户端存储使用信息
    clients: {
      // 客户端(host_123456)
      "host_123456": {
        // key值：客户端ident；value值：客户端存储使用量(GB)

        "storage_abcdef": 0,
      },
      "host_abcdef": {
        "storage_123456": 120,
        "storage_abcdef": 10,
```

### 4、最近7天整机备份数据量

- url: `https://<IP地址已脱敏>:80/apiv1/v1/backup/history/?token=<已脱敏>
- method: GET
- params:
    - startDate: 开始日期，闭区间
    - endDate: 结束时间，闭区间
- response:

```json5
{
  "message": "获取成功",
  "errcode": 0,
  "data": {
    // 最近7天所有客户端新增数据量(日期顺序递增，共7条数据)
    "storages": [
      // 表示在日期2020-01-22所有客户端新增数据量
      {
        "date": "2020-01-22",
        // key值：客户端ident；value值：客户端新增数据量(GB)
        "delta": {
          "host_123456": 4,
          "host_abcdef": 3,
          ...
        }
      },
      // 表示在日期2020-01-23所有客户端新增数据量
      {
        "date": "2020-01-23",
        "delta": {
          "host_123456": 24,
          "host_abcdef": 13,
          ...
        }
      },
      ...
    ],
  }
}
```

### 5、警告统计

- url: `https://<IP地址已脱敏>:80/apiv1/v1/backup/alters/?token=<已脱敏>
- params:
    - index：数据索引号(表示请求大于该索引号的所有数据；建议传递：上一次请求获取到的末尾数据的索引号）
- method: GET
- response:

```json5
{
  "message": "获取成功",
  "errcode": 0,
  "data": {
    // 告警信息(index递增)
    "alter_msg": [
      {
        "index": 12,
        // 告警发生的时间
        "time": "2021-01-23 13:33:12",
        // 告警内容
        "content": "客户端：oracle1,CDP 暂停，原因：客户端离线"
      },
      {
        "index": 13,
        // 告警发生的时间
        "time": "2021-01-23 14:43:10",
        // 告警内容
        "content": "客户端：oracle1备份失败，原因：网络链接超市离线"
      },
      ...
    ]
  }
}
```
