# 业务维护资料整理

本目录用于承接 `D:\新建文件夹\业务维护清单` 中已有的业务系统、平台、基础设施和运维资料。

## 整理边界

本次整理只基于本地可见的目录、文件名、文件类型、文件大小和修改时间完成，未连接任何服务器、接口或业务系统，也未做任何连通性、登录、功能或恢复测试。

当前部分 Excel / Word 文件处于云文件占位状态，读取正文时系统返回云文件提供程序不可用。因此本次不会把表格内容、访问地址、账号、密码或端口明细写入知识库正文。

## 产物

- [business-system-inventory.md](business-system-inventory.md)：业务系统和平台台账初版。
- [operation-domains.md](operation-domains.md)：按运维域归类的资料视图。
- [source-material-index.md](source-material-index.md)：原始资料索引和敏感资料处理建议。

## 使用方式

1. 先由系统负责人确认台账中的系统归类。
2. 对核心系统逐个补充负责人、影响范围、依赖关系和升级路径。
3. 对含账号、密码、端口、访问地址的文件，只在受控位置保存，不在普通知识库中展开。
4. 工单系统推荐知识时，优先使用系统名称、运维域和问题类型进行匹配。
