# 运维域分类

本文件用于把业务维护资料按运维域组织，便于后续工单推荐和知识库导航。

## 生产制造与车间系统

- MES
- WMS
- 标准车间立库海康系统
- 蒲瑞立库
- 套料系统

常见工单入口：

- MES 登不上
- 车间终端无网络
- 立库系统异常
- 打印标签或标识卡异常
- 生产辅助软件无法启动

## ERP、财务与供应链

- S4
- HANA
- NC
- U8
- SRM
- PO
- 百望云数电
- 税控系统

常见工单入口：

- 系统登录失败
- 接口异常
- 报表或单据异常
- 客户端或控件异常
- 外部税务平台或发票服务异常

## 协同办公与研发管理

- OA系统
- PLM
- 人力
- 技术中心
- 集团官网
- 云盘
- 邮件代理
- RTX

常见工单入口：

- OA 无法登录或附件上传失败
- PLM 客户端异常
- 云盘同步或共享权限异常
- 邮件收发异常
- 官网证书、域名或发布异常

## 身份认证与终端安全

- AD
- KMS
- DLP
- 桌管
- 堡垒机
- JumperServer

常见工单入口：

- 域账号登录失败
- Windows / Office 激活失败
- 终端策略拦截
- 堡垒机授权或审计异常
- 桌面管控客户端异常

## 基础设施、存储与备份

- vSphere
- 深信服超融合
- 深信服分布式存储
- 分布式存储
- 存储系统
- 备份系统
- 服务器日志收集
- 机房设备说明书

常见工单入口：

- 虚拟机异常
- 存储容量告警
- 备份失败
- 恢复需求
- 日志采集异常
- 机房设备告警

## DevOps 与制品管理

- Maven / Nexus

常见工单入口：

- 构建依赖下载失败
- 制品库权限异常
- 代理源异常
- 仓库存储容量不足
